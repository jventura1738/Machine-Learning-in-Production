# open-event-scripts

Useful scripts for managing events using Open Event API

## Instructions

```sh
poetry install
```

```sh
poetry shell
```

OR 

Using pip

```sh
pip install -r requirements.txt
```

## Usage

```sh
python scripts/reschedule.py <event_identifier>
```

```sh
python scripts/generate_session_tweet_csv.py <event_identifier>
```
