# betterspecs.org

Source code for [alpha.betterspecs.org](https://alpha.betterspecs.org)

## Requirements

RSpec Best Practices website is tested against MRI 2.7.0


## Installation

    git clone git@github.com:betterspecs/betterspecs.git
    cd betterspecs
    bundle install
    bundle exec jekyll serve
