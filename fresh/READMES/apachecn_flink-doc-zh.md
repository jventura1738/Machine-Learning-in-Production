## 下载

### Docker

```
docker pull apachecn0/flink-doc-zh
docker run -tid -p <port>:80 apachecn0/flink-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install flink-doc-zh
flink-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flink-doc-zh
flink-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
