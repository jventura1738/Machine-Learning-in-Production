## build from source

```bash
git clone git@github.com:crimx/ext-saladict.git
cd ext-saladict
yarn install
yarn pdf
```

Add a `.env` file following the `.env.example` format(leave empty if you don't use these dictionaries).

```bash
yarn build
```

Artifacts can be found in `build/`.
