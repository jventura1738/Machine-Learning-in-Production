# FUSEBOX v4 is out!

Install:

```
npm install fuse-box --save-dev
```

```ts
import { fusebox } from 'fuse-box';
fusebox({
  target: 'browser',
  entry: 'src/index.tsx',
  webIndex: {
    template: 'src/index.html',
  },
  devServer: true,
}).runDev();
```
