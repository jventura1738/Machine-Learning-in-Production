### Demo

以下为一个简单日历展示的使用方法

``` json {3}
{
  "usingComponents": {
    "calendar": "/your_component_path/calendar/index"
  }
}
```

在页面 `wxml` 中引入组件

```xml
<calendar />
```

更多自定义配置及功能 [请查阅日历文档](https://treadpit.github.io/wx_calendar)
