## 下载

### Docker

```
docker pull apachecn0/spark-doc-zh
docker run -tid -p <port>:80 apachecn0/spark-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install spark-doc-zh
spark-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g spark-doc-zh
spark-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
